DTox

An educational application exposing commonly found toxic chemicals within everyday products.

MERN stack, Google Firebase, Heroku, Netlify, Sass, Materialize

  • Engineered a RESTful backend to manipulate the names, descriptions and images of everyday products with Mongoose, Express and Node
  • Used React hooks so user-created “healthy alternative products” control the form and handle submissions
  • Authorized users with OAuth via Google Firebase to perform CRUD operations
  • Created a compelling user interface using MaterialUI components augmented with Sass
  • Implemented a maintainable and responsive layout with CSS grid and flexbox
  • Deployed the backend with Heroku and the frontend with Netlify
